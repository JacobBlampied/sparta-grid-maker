#SPARTA GRID MAKER

##DESCRIPTION
The aim of this program is to generate a grid on a blank page with the dimensions specified from user input via a prompt at the start of the program. when clicked, the box should change colour.

##LANGUAGES
HTML, CSS, JavaScript

##HOW TO DOWNLOAD
